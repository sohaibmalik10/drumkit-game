## Usage

Click on the drum elements with your mouse to trigger sounds.
Alternatively, use the <br> keyboard keys </br> corresponding to each drum element for a hands-free experience.
"w" for crash
"a" for kick bass
"s" for snare
"d" for tom 1
"j" for tom 2
"k" for tom 3
"l" for tom 4
